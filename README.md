# CI/CD Jenkins Pipelines

This repository contains multiple Jenkins pipelines for CI/CD practice.

##  Contents
- Script Runner Pipeline
- Parameterized Pipeline
- Docker Build & Deploy Pipeline
- Docker Build & Push Pipeline

##  How to use

1. Create a Jenkins job
2. Select "Pipeline from SCM"
3. Add this repository URL:
   https://github.com/MariumTanious/marium
4. Choose branch: main
5. Set Script Path to the required pipeline file

##  Files
- script-runner.groovy
- parameterized-docker-pipeline.groovy
- docker-build-push-pipeline.groovy
- run.sh

##  Requirements
- Jenkins installed
- Docker installed (for Docker pipelines)
- GitHub integration configured
